## About

fex is a keyboard driven multi-tab **f**ile **e**xplorer for UNI**X** like systems written in C

For usage and keybinds take a look at `config.h` 

## Build-time dependencies
- C compiler
- make

## Installing
```
git clone https://github.com/tranarchy/fex
cd fex
make
sudo make install
```
